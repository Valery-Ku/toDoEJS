# ToDo Приложение

Это простое веб-приложение для управления задачами (ToDo), разработанное на **Node.js**, **Express**, **MongoDB** и **EJS**. Оно позволяет создавать, просматривать, редактировать и удалять задачи, а также отмечать их как выполненные.

## Функциональность

- **Создание задач**: Добавляйте новые задачи с заголовком и описанием.
- **Просмотр задач**: Список всех задач с возможностью фильтрации по статусу (выполненные/невыполненные).
- **Редактирование задач**: Изменяйте заголовок и описание существующих задач.
- **Удаление задач**: Удаляйте задачи из списка.
- **Отметка выполненности**: Ставьте галочку, чтобы пометить задачу как выполненную.
- **Адаптивный дизайн**: Приложение работает на десктопе и мобильных устройствах с плавными анимациями и стилизацией.

## Требования

- **Node.js** (версия 14 или выше)
- **MongoDB** (локально или в облаке, например, MongoDB Atlas)
- **npm** (для управления зависимостями)

## Установка

1. **Клонируйте репозиторий**:
   git clone <URL_репозитория>
   cd todo-app

2. **Установите зависимости**:
   npm install

3. **Настройте базу данных**:
   - Убедитесь, что MongoDB запущена локально (по умолчанию на `mongodb://localhost:27017/todoapp`) или укажите URL в переменной окружения.
   - Если используете MongoDB Atlas, создайте кластер и получите connection string.

4. **Настройте переменные окружения** (опционально):
   - Создайте файл `.env` в корне проекта:
     MONGODB_URI=mongodb://localhost:27017/todoapp
     PORT=3000
   - Если не указано, приложение использует значения по умолчанию.

## Запуск

1. **Запустите сервер**:
   npm start

2. **Откройте браузер**:
   Перейдите по адресу `http://localhost:3000` (или указанному порту).

## Использование

- **Главная страница**: Просмотр списка задач. Нажмите на чекбокс, чтобы отметить задачу как выполненную.
- **Добавление задачи**: Нажмите "Добавить задачу" и заполните форму.
- **Редактирование**: Нажмите "Редактировать" рядом с задачей, измените данные и сохраните.
- **Удаление**: Нажмите "Удалить" рядом с задачей для подтверждения удаления.

Приложение автоматически сохраняет изменения в MongoDB.

## Зависимости

- `express`: Фреймворк для Node.js.
- `mongoose`: ODM для MongoDB.
- `ejs`: Шаблонизатор для рендеринга HTML.
- `method-override`: Для поддержки PUT/DELETE в формах HTML.
- `dotenv`: Для переменных окружения.
